Operating System Terminology

Loren Lemarr

TERM DEFINITION
operating system An operating system is the software that controls the overall operation of a computer. This includes file management, the user interface, and the program execution enviornment.
job queue A job queue is a storage organization in which jobs are ordered in first-in, first-out fashion.
FIFO FIFO is the abreviation for first-in, first-out and means that jobs are removed from the queue in the order in which they arrived.
interactive processing An operating system that allowed a program being executed to carry on a dialog with the user through remote terminals.
time-sharing A feature of operating systems that let those systems provide service to multiple users at the same time.
multiprogramming A technique for implementing time-sharing that divides timen into intervals and then the execution of each job is restricted to only one interval at a time.
application software Application software consists of the programs for performing tasks particular to the machine's utilization.(spreadsheets, satabase systems, desktops publishing systems, program development ssoftware, and games)
system software system software performs those tasks that are common to computer systems in general. It provides the infrastructure that application software requires.
utility software One of the catagories of system software consists of programs for performing activities that are fundamental to computer instalation but no included in the operating system.
graphical user interface(GUI) User interface that represents objects that can be manipulated, such as files and programs, pictorially on the display as icons.
shell A user interface that commincated with users through textual messages using a keyboard and monitor screen.
kernel The internal part of an operating system that contains software components that perform the very basic functions required by the computer intallation.
device drivers Components of the kernal which are the software units that communicate with the controllers (or at times directly with peripheral devices) to carry out operations on the peripheral devices attached to the machine.
file manager Component of the kernal, whose job is to coordinate the use of the machine's mass storage facilities. Maintains records of alll the files stored in mass storage and where they are located.
memory manager Component of the kernal that coordinates the machine's use of main memory.
virtual memory The illusion of additional storage space created by rotating programs and data back and forth between main memory and mass storage. Lets the computer function as if it had more memory space.
boot strapping Procedure in which the operating system gets started that is performed each time it is turned on. Transfers the operating system from mass storage into main memory.(Booting up your computer)
ROM The abreviation for read-only memory which are non volitile memory cells in the CPU that can be read but not altered. This allows the CPU to get the operating system from the mass storage.
deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
spooling The technique of holding data for output at a later but more convenient time.(a deadlock avoidence scheme)
super user (or administrator) An account that gains highly priviliged access to the opperating system by identifying themselves as the administrator during the login procedure. The administrator can alter settings within the opperating system, modify critical software packages, adjust the priviliges granted to other users, and other maintenance activities.
multi-factor authentication When more than one piece of evidence is required to gain access. Such as requiring a special access code along side the users password.
